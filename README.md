# CrimeSpot-api

## API key

For the program to run properly, a config.js file must be created locally in your project.

config.js will contain the following code: 

```node
const config = {
  KEY: [API KEY],
};

module.exports = config;
```
replace [API KEY] with a valid api key

module.exports = config;

## Run locally

```node
npm install
```

```node
node app.js
```
